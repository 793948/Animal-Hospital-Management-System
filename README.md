# Animal-Hospital-Management-System
## 📌 Project Description
This project is a web-based application that helps users report injured animals and get medical assistance. It uses AI to analyze injury images and suggests appropriate hospitals and doctors.

## 🎯 Features
- Upload image of injured animal
- AI-based injury detection
- Doctor recommendation
- Nearby hospital locator
- Emergency alert system

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Database: SQLite
- AI/ML: TensorFlow / Keras

## ▶️ How to Run
1. Install Python
2. Install required libraries:
   pip install flask tensorflow opencv-python
3. Run:
   python app.py
4. Open browser and go to:
   http://127.0.0.1:5000/
